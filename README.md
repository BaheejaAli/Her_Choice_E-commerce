# Her Choice – E-Commerce Website 🛍️

Her Choice is a modern E-Commerce web application built using **Django**, **PostgreSQL**, and **Tailwind CSS**.  
The project follows clean architecture, custom authentication, and a scalable folder structure suitable for large production-level applications.

---

## 🚀 Features

- Custom User Model (extendable for phone/email login)
- User Registration & Login System
- PostgreSQL Database Integration
- Environment variables using `.env`
- Responsive UI using Bootstrap CSS
- Scalable and modular project structure
- Git Branch Workflow:
  - `main` (stable)
  - `develop` (active development)
  - `feature/*` (individual features)
